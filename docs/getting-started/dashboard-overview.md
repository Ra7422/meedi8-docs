# Dashboard Overview

Your dashboard is your home base in Meedi8. Here's what you'll find.

## Main Sections

### Sessions Dashboard

The main view shows your mediation sessions:

- **Active Sessions** - Ongoing mediations
- **Pending** - Waiting for the other person to join
- **Completed** - Finished sessions with reports

### Navigation

- **Home** - Return to dashboard
- **New Session** - Start a new mediation
- **Settings** - Account and subscription
- **Help** - Access documentation

---

## Session Cards

Each session shows:

| Element | Description |
|---------|-------------|
| **Title** | Session topic/description |
| **Status** | Active, Pending, or Completed |
| **Participants** | Who's involved |
| **Last Activity** | When last updated |
| **Actions** | Continue, View Report, or Delete |

### Status Indicators

- **Active** - Session in progress, you can continue
- **Pending Invite** - Waiting for partner to join
- **Your Turn** - Partner is waiting for your response
- **Their Turn** - Waiting for partner's response
- **Completed** - Session finished, report available

---

## Quick Actions

### Start New Session

Click **"New Session"** to begin:
1. Choose **Solo** (just you) or **Duo** (with someone)
2. Describe the situation
3. Begin coaching

### Continue Session

Click on any active session card to:
- Resume the conversation
- Check status
- View progress

### View Report

Completed sessions have a **"View Report"** button:
- Download PDF summary
- Review key insights
- See agreed actions

---

## Settings

Access Settings to:

- **Profile** - Update name, email preferences
- **Subscription** - Upgrade or manage plan
- **Notifications** - Email and push settings
- **Privacy** - Data and account options
- **Logout** - Sign out securely
