# Importing Telegram Chat History

Give Meedi context by importing your Telegram conversations.

## Why Import Chat History?

Importing conversations helps Meedi:
- Understand the full context
- See communication patterns
- Provide more relevant guidance
- Reference specific incidents

---

## How to Import

### Step 1: Export from Telegram

1. Open Telegram Desktop
2. Go to the chat you want to export
3. Click the three dots (menu)
4. Select **"Export Chat History"**
5. Choose **JSON format**
6. Save the file

### Step 2: Upload to Meedi8

1. Start a new session
2. Click the **Telegram icon** or **"Import Chat"**
3. Select your exported JSON file
4. Choose which messages to include

### Step 3: Review Selection

You can:
- Select specific date ranges
- Choose specific conversations
- Preview what will be imported
- Exclude sensitive content

---

## What Gets Imported

| Included | Not Included |
|----------|--------------|
| Text messages | Deleted messages |
| Timestamps | Your password |
| Sender names | Other chats |
| Replies | Media files* |

*Images and files are referenced but not stored on our servers.

---

## Privacy Considerations

### Your Data
- Imported messages are encrypted
- Only you and your session partner see them
- Data can be deleted anytime

### Other Person's Messages
- Their messages are included (for context)
- They'll know you imported chat history
- Used only for mediation purposes

---

## Tips for Best Results

:::tip Select Relevant Periods
Focus on the time period related to your conflict.
:::

:::tip Include Context
Include messages before the issue started for better understanding.
:::

:::tip Be Selective
You don't need to import everything - just what's relevant.
:::

---

## Troubleshooting

### File Won't Upload?
- Make sure it's JSON format
- Check file size (max 50MB)
- Try a different browser

### Messages Not Showing?
- Verify the export completed
- Check date range selection
- Refresh and try again
