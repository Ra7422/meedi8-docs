# Voice Messages

Sometimes it's easier to talk than type. Meedi supports voice messages.

## How It Works

1. **Record** your message using the microphone button
2. **Meedi transcribes** your audio using AI (OpenAI Whisper)
3. **Review** the transcription
4. **Meedi responds** to your message

---

## Recording a Message

### Step 1: Click the Microphone
Look for the microphone icon in the message input area.

### Step 2: Speak Clearly
- Speak naturally
- Minimize background noise
- Keep messages under 2 minutes

### Step 3: Stop Recording
Click the stop button when finished.

### Step 4: Review
You'll see the transcription. You can:
- Send as-is
- Edit the text
- Re-record if needed

---

## Tips for Clear Audio

:::tip Quiet Environment
Record in a quiet space for best transcription accuracy.
:::

:::tip Speak Naturally
Talk as you would normally - no need to slow down artificially.
:::

:::tip One Topic at a Time
Focus on one point per message for clearer communication.
:::

---

## Privacy

- Audio is processed securely
- Transcription happens via encrypted API
- Original audio is not stored long-term
- Only the transcript remains in your session

---

## Browser Permissions

You'll need to allow microphone access:

### Chrome
Click the lock icon → Site settings → Microphone → Allow

### Safari
Safari → Settings → Websites → Microphone → Allow

### Firefox
Click the lock icon → Permissions → Allow microphone

---

## Troubleshooting

### Microphone Not Working?
1. Check browser permissions
2. Test microphone in another app
3. Try a different browser

### Poor Transcription?
- Reduce background noise
- Speak more clearly
- Try shorter messages

### Audio Not Recording?
- Refresh the page
- Check if another app is using the mic
- Try a different device
