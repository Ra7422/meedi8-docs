# File Attachments

Share documents and images to give Meedi more context.

## Supported File Types

| Type | Formats | Max Size |
|------|---------|----------|
| Images | JPG, PNG, GIF, WebP | 10MB |
| Documents | PDF, DOC, DOCX | 10MB |
| Text | TXT, CSV | 5MB |

---

## How to Attach Files

### During a Session

1. Click the **+ button** next to the message input
2. Select your file
3. Wait for upload to complete
4. Continue the conversation

### What Meedi Does

For **images**, Meedi can:
- Analyze visual content
- Read text in images (OCR)
- Describe what it sees

For **documents**, Meedi can:
- Read and summarize content
- Reference specific sections
- Answer questions about them

---

## Use Cases

### Screenshots
Share screenshots of:
- Text conversations
- Emails
- Social media posts

### Documents
Share relevant:
- Contracts or agreements
- Emails
- Written communications

### Images
Share photos related to:
- Situations being discussed
- Evidence or context
- Anything relevant

---

## Privacy & Security

- Files are encrypted during upload and storage
- Only visible to session participants
- Automatically deleted after account deletion
- Never shared with third parties

---

## Tips

:::tip Label Your Files
Briefly explain what you're uploading so Meedi can use it effectively.
:::

:::tip One at a Time
Upload files one by one for better processing.
:::

:::tip Relevant Only
Only attach files that add meaningful context.
:::

---

## Troubleshooting

### Upload Failed?
- Check file size (max 10MB)
- Verify file type is supported
- Check internet connection

### Meedi Can't Read File?
- Try a different format
- Ensure image is clear
- Convert to PDF if needed
