# Understanding Your Duo Report

After a duo mediation, both participants receive a shared report.

## Report Sections

### Session Summary
A neutral overview of:
- What was discussed
- Key points raised by each person
- Areas of agreement and difference

### Both Perspectives
Summaries of each person's:
- Main concerns
- Feelings expressed
- Underlying needs

### Common Ground
Areas where you both:
- Agreed
- Found understanding
- Shared similar needs

### Areas for Growth
Topics that may need:
- More discussion
- Outside support
- Time to process

### Agreed Actions
Specific commitments made:
- What each person will do
- When/how to follow up
- Next steps

### Recommendations
Meedi's suggestions for:
- Maintaining progress
- Preventing future conflicts
- Resources that might help

---

## Sharing the Report

Both participants receive:
- **Same report** - Identical content
- **Download option** - PDF format
- **Email copy** - Sent to both

### What's Private
Your individual coaching sessions remain private. The report only includes:
- Joint conversation content
- Agreed summaries
- Shared commitments

---

## Using the Report

### For Accountability
Reference the agreed actions to:
- Track commitments
- Hold each other accountable
- Measure progress

### For Follow-Up
Schedule a check-in to:
- Discuss how things are going
- Revisit any issues
- Celebrate improvements

### For Documentation
Keep the report for:
- Personal records
- Future reference
- Evidence of effort

From your completed session:

---

## If You Disagree with the Report

The report reflects what was discussed. If something seems inaccurate:
- It may reflect a miscommunication
- Consider discussing with your partner
- Contact support if there's a technical issue

---

## Next Steps After Mediation

### Schedule Follow-Up
Consider a check-in after:
- 1 week - Initial progress
- 1 month - Sustained change
- As needed - Address new issues

### Continue the Work
Consider:
- Individual counseling
- Couples/relationship therapy
- Communication skills training

### Return to Meedi8
You can always start new sessions for:
- New topics
- Follow-up on old issues
- Ongoing support
